# Security Policy

## Supported Versions

This project is maintained on a best-effort basis. Only the latest version is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Instead, please email the maintainer at [sikienzl.github@t-online.de] with details.  
We will respond as soon as possible and coordinate a fix.

Thank you for helping keep this project and its users safe!
