# Contributing

Thank you for considering contributing to this project!

## How to Contribute

- Fork the repository and create your branch from `master`.
- Make your changes.
- Ensure your code is clean and well-documented.
- If adding a feature or fixing a bug, please add or update tests if possible.
- Submit a pull request with a clear description of your changes.

## Code Style

- Use clear, descriptive commit messages (max 80 characters per line).
- Follow PEP8 for Python code style.
- Keep code readable and add comments where helpful.

## Reporting Issues

- Use the issue tracker to report bugs or suggest features.
- Provide as much detail as possible (steps to reproduce, logs, etc.).

## Security

If you discover a security vulnerability, please see `SECURITY.md` for how to report it.

Thank you for helping to improve this project!
